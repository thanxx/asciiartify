1. After creating an application in ArgoCD port-forward to `svc/ambassador`
    
    ```
    k port-forward -n demo svc/ambassador 8080:80
    ```

2. Open another instance of the terminal, and check if the service returns the version:
   ```
    curl  localhost:8080
    k8sdiy-api:599e1af
   
3. Download image
    ```
    curl https://logonoid.com/images/volt-logo.png -o logo.png

4. Upload the image to the service and get an ASCII-converted image:
   ```
   curl -F 'image=@logo.png' localhost:8080/img/
